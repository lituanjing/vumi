## 介绍

一个基于 Vue.js 2.x 的 UI 组件库

## 作者

Email: tuanjing@outlook.com

## 开始使用

1. 安装
    ```bash
    yarn add vumi
    ```
2. 引入 vumi
    ```javascript
    import "../node_modules/vumi/dist/index.css"
    import { YmButton } from 'vumi'

    export default {
      components: { YmButton }
    }
    ```
